Game overview:
    Gomoku
TargetPlatform:
    Both mobile and desktop
Game Genre:
    Chess
Game objective:
    The winner is the first player to form an unbroken line of five stones of their color
    horizontally, vertically, or diagonally.
Game Mechanics:
    This game has two players,
    In each turn, players take turns placing their markers on an empty intersection point on the board.
    Black plays first.
    Wining condiction:
        The goal is to be the first to create an unbroken row of five consecutive markers
        horizontally, vertically, or diagonally.
    draw condiction:
        If the board is filled before either player achieves five in a row, the game is a draw.
